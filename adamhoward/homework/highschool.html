HONORS COURSES

Courses which are part of the honors program at Thomas More cover more difficult and in greater depth material. A higher quality of work is expected and there is a regular course against which to compare the course. Departments which include honors course must re-evaluate those course with the Assistant Principle for Academic Affairs yearly.

Honors courses which include Geometry Honors, Algebra II/Trig Honors, Calculus, English II Honors, English III Honors, English IV Honors, Biology Honors, Chemistry Honors, and Physics Honors will be weighted for determining Grade Point Average, according to the following system.

A+ = 5.0B+ = 4.0C+ = 3.0D+ = 2.0
A = 4.7B = 3.7C = 2.7D = 1.7
A- = 4.3B- = 3.3C- = 2.3D- = 1.3
